# WeatherApp

WeatherApp is a lightweight PyQt6 desktop weather viewer.

Primary goals:

- Provide a fast, simple desktop interface for weather information
- Display current conditions, hourly forecast, and daily forecast
- Remain responsive by performing network operations in background workers
- Maintain minimal dependencies and clear architecture

Target platform:
Linux desktop environments (tested on Linux Mint 22.3).

Core features (planned):

1. Current weather conditions
2. 48-hour hourly forecast
3. 7–10 day daily forecast
4. City search
5. Manual refresh

Design philosophy:

- Simple UI over visual complexity
- Clear separation between data fetching, formatting, and GUI
- Maintainable Python code following PEP 8
